Fonctionnalités

Tout ce dont vous avez besoin

Une suite complète d'outils RH pour piloter votre organisation avec efficacité.

Gestion des Employés
CRUD complet avec filtrage par département, recherche, pagination et export CSV/Excel/PDF. Profils détaillés avec toutes les informations.
Dashboard Analytics
5 graphiques interactifs (lignes, barres) pour visualiser performances, présences et évolution des effectifs en temps réel.
Suivi de Présence
Marquage présent/absent par employé avec liste quotidienne, statuts dynamiques et historique consultable à tout moment.
Départements & Rôles
Structurez votre organisation en départements et assignez des rôles professionnels précis à chaque collaborateur.
Performances
Évaluez et suivez les performances individuelles avec des fiches détaillées, historiques et indicateurs clés par employé.
Rapports & Exports
Générez des rapports détaillés et exportez vos données en CSV, Excel ou PDF pour vos besoins de reporting.

Sécurité

Protection de niveau entreprise

Chaque couche de l'application est sécurisée pour protéger vos données et vos accès.

JWT Authentication
Tokens signés, expiration 1h, vérifiés à chaque requête protégée.
Helmet.js + Rate Limiting
Headers HTTP sécurisés et 100 req/15min max pour prévenir les attaques.
RBAC — Contrôle des rôles
Accès aux routes restreint selon le rôle : admin ou manager uniquement.
middlewares/authenticateToken.js
// Vérification du token JWT
const authenticateToken = (req, res, next) => {
  const authHeader = req.headers['authorization'];
  const token = authHeader?.split(' ')[1];

  if (!token) return res.sendStatus(401);

  jwt.verify(token, process.env.JWT_SECRET, (err, user) => {
    if (err) return res.sendStatus(403);
    req.user = user;
    next();
  });
};

// Contrôle des rôles
const authorizeRoles = (...roles) => {
  return (req, res, next) => {
    if (!roles.includes(req.user.role))
      return res.sendStatus(403);
    next();
  };
};
